การเชื่อมต่อฐานข้อมูลล้มเหลว (Connection Failed)

ปัญหานี้เกิดจาก **Firebase Security Rules** ของคุณยังไม่ถูกต้อง

วิธีแก้ไข (โปรดทำตามอย่างละเอียด):

  1. ไปที่โปรเจกต์ของคุณใน **Firebase Console**
  2. ไปที่เมนู **Build > Realtime Database > Rules**
  3. **ลบโค้ดเก่าทั้งหมด** และวางโค้ดนี้ลงไป: 
    {
  "rules": {
    ".read": true,
    ".write": true
  }
}
  4. กดปุ่ม **PUBLISH** สีฟ้าเพื่อบันทึก
  5. กลับมาที่หน้าเว็บนี้แล้ว **รีเฟรช (กด Ctrl+Shift+R)**
